# AegisAIS — Pricing Model

## Pricing Strategy

Annual license tiers by vessel monitoring capacity. Aligned with port authority procurement budgets and infrastructure operator requirements.

## Tiers

| Tier         | Annual Price  | Vessel Capacity           | Includes                                                                                     |
| ------------ | ------------- | ------------------------- | -------------------------------------------------------------------------------------------- |
| **Coastal**  | €60,000/year  | Up to 100 tracked vessels | Core anomaly detection (7 rules), alert dashboard, track replay, monthly reports             |
| **Maritime** | €120,000/year | Up to 500 tracked vessels | Coastal + geofence monitoring, watchlists, incident tracking, API access, Prometheus metrics |
| **Fleet**    | Custom        | 500+ vessels              | Maritime + custom detection rules, multi-zone deployment, dedicated support, SLA             |

## Add-Ons

| Add-On                       | Price        | Description                                                             |
| ---------------------------- | ------------ | ----------------------------------------------------------------------- |
| Baltic Cable Geofence Module | €20,000/year | Pre-configured geofence zones for Nordic submarine cable infrastructure |
| Historical Replay Archive    | €15,000/year | 12-month track history retention with replay and export                 |
| Custom Integration           | From €10,000 | STIX/TAXII feed integration, national AIS data source connection        |

## Procurement-Friendly Packaging

- Annual invoicing (net 30 terms standard for government)
- Multi-year discount: 10% for 3-year commitment
- Pilot tier available: 90-day evaluation at 50% of Coastal rate (€30,000)
- All pricing excludes VAT; government entities exempt where applicable
